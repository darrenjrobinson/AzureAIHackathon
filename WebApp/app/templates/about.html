About Owner Builder AI Assistant

A specialised tool for Owner Builders in Australia

What is an Owner Builder?

In Australia, an Owner Builder is someone who obtains a permit to carry out residential building work on their land, taking on the responsibilities of a builder. This includes managing, coordinating and supervising the construction work.

How This Tool Helps

The Owner Builder AI Assistant provides three main features to help you manage your project:

  • AI Prompt Generator: Creates tailored prompts to help you get the information you need for specific building tasks.
  • Document AI Search: Searches through relevant building documentation to answer your questions based on your generated prompt.
  • AI Drawing Analyzer: Uses AI to analyze your construction drawings and provide insights.

Technologies Used

This application is built using:

  • Flask - Python web framework
  • Azure AI Services - For intelligent document analysis
  • Azure AI Search - For intelligent document search
  • Azure OpenAI - For generating responses and analysis
  • Azure Computer Vision - For analyzing construction drawings
